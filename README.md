## Prerequisites

Have Node.js installed on your machine. (https://nodejs.org/en/download/)

## Running the server

From root of repository:

1.  `cd server`
2.  `npm install`
3.  `nodemon index.js`

## Running the client

From root of repository:

1.  `cd client`
2.  `npm install`
3.  `npm start`
4.  To view and interact with the web application, open browser and proceed to `http://localhost:3000`
